# Skills

Claude Code slash commands for Measure workflows.

## How to use

Symlink or copy individual skill files into `~/.claude/skills/`:

```bash
ln -s /path/to/measure-ai-tools/skills/create-insight-card.md ~/.claude/skills/create-insight-card.md
```

Then invoke with `/create-insight-card` in any Claude Code session.

## Skills

| Skill | Command | Description |
|-------|---------|-------------|
| Create Insight Card | `/create-insight-card` | Transforms raw data and context into a formatted Insight Card JSON blob ready to upload to Studio |
